I am a front-end web developer based in San Diego, CA. I have a Bachelor's of Science in Interactive Media Design, which translates to "I build web applications."

Straight Priorities

  • Usability First: Web applications are tools first. If you can't use it, no amount of bells and whistles are going to fix it.
  • Flexible Development: Design changes and code will break. Developing flexible products ensures longevity and painless changes down the road.
  • Give It A Soul: Style and presentation go a long way. Setting the right tone enhances experiences and can be the difference between good and blow-your-pants-off awesome.
